Creación de una Página Web de Votos

Este proyecto se enfocó en crear una página web para gestionar los votos en la elección de un rector en la Universidad Técnica de Ambato (UTA). La web permitía a los usuarios ver los resultados de los votos por partido político en tiempo real. Se utilizaron tecnologías como HTML, PHP para el backend y SQL para gestionar la base de datos. Además, el diseño incluyó elementos visuales como un fondo en movimiento y gráficos atractivos, mejorando la experiencia del usuario.

Creación de un Tamagochi en Python

El Tamagochi fue un proyecto interactivo donde los usuarios cuidaban una mascota virtual. Utilizando Java, se implementó un sistema con indicadores de salud y felicidad de la mascota, y el usuario debía alimentarla y atenderla. El proyecto utilizó conceptos de programación orientada a objetos para crear la lógica del juego y almacenó el estado del Tamagochi en variables.

Creación de una Plataforma Educativa

Este proyecto consistió en desarrollar una plataforma educativa, donde los usuarios podían acceder a contenido académico, subir trabajos, revisar sus calificaciones, etc. Se implementaron funciones de carga de material educativo, gestión de usuarios, y evaluación de tareas. PHP y JavaScript fueron utilizados para crear la funcionalidad interactiva de la plataforma, mientras que una base de datos MySQL se encargó de la gestión del contenido y los registros de los usuarios.

Creación de un Blog

El proyecto consistió en desarrollar un blog donde los usuarios podían publicar contenido libremente, desde texto hasta imágenes. El sistema de gestión de usuarios permitía a los usuarios crear, editar y eliminar sus publicaciones. Utilizando PHP y MySQL, se implementaron las funcionalidades básicas de un blog, como la gestión de posts y la interacción con la base de datos, proporcionando una plataforma sencilla y accesible para todos los usuarios.
